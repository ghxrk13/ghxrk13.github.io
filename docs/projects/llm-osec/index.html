llm-osec

OWASP LLM Top 10 scanner — a CLI that probes a deployed LLM application for the top categories of risk and produces a structured findings report.

Status: in development Python · Claude SDK · CISSP-flavored

Why this exists

Most security tooling for traditional applications (SAST, DAST, fuzzers) does not catch the failure modes specific to LLM-powered applications: prompt injection, sensitive data leakage through hallucination, insecure output handling, training-data poisoning, and the other risks documented in the OWASP Top 10 for LLM Applications. Security teams adopting LLM features are mostly hand-rolling tests or pulling pieces from research repos.

llm-osec is a focused, opinionated scanner that runs a library of probes against a target LLM endpoint and writes a structured report (severity, category, evidence, suggested mitigation). The goal is something a CISSP / AppSec engineer can drop into a CI pipeline or run on demand, not a research toolkit that requires a PhD to operate.

What it covers (MVP scope)

  • LLM01 — Prompt Injection. Direct + indirect injection probes; system-prompt exfiltration attempts.
  • LLM02 — Insecure Output Handling. XSS / SSRF / injection vectors smuggled through model output.
  • LLM06 — Sensitive Information Disclosure. Memory / cache probing; system-prompt leakage.
  • LLM08 — Excessive Agency. Tool-use boundary probes when the target exposes function calling.
  • LLM10 — Model Theft. Output-distillation rate-limiting checks.

Approach

  • Probe library encoded as Markdown with metadata + expected-failure signatures — easy to extend, easy to audit.
  • Claude as oracle for evaluating ambiguous probe responses (signature match alone is too brittle for prompt-injection success).
  • Pluggable target backends: any HTTP endpoint that takes JSON in / JSON out, with adapters for Anthropic / OpenAI / Ollama / generic REST.
  • Report output: JSON (machine), Markdown (human), and a printable HTML one-pager.

Where it goes

Open-source on GitHub. Becomes the foundation of a productized AI-security consulting offering once the W-2 transition is complete and contract constraints clear.
